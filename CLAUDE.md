- **Secrets:** keys live in `.env.local` only (git-ignored via `*.local`); Vercel env vars via Dashboard. Never print key values.

- **Verification:** browser preview tooling does not work in this environment â€” verify with `tsc -b` + `vite build` + live curl checks against prod APIs; UI interactions get eyeballed by Shamil.
